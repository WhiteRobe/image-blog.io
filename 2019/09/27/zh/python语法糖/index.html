Python装饰器的三层结构

  1. 第一层(装饰器包裹器)：接受装饰器的新参数，返回装饰器
  2. 第二层(装饰器)：接受被包裹的函数，返回内部包裹器
  3. 第三层(内部包裹器)：接受参数，实现新业务逻辑
